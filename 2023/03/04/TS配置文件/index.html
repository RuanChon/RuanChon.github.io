tsconfig.json

通过 tsc init 生成或者脚手架初始化


配置说明

  1. include - 指定编译文件默认是编译当前目录下所有的 ts 文件

  2. exclude - 指定排除的文件

  3. target - 指定编译 js 的版本例如 es5 es6

  4. allowJS - 是否允许编译 js 文件

  5. removeComments - 是否在编译过程中删除文件中的注释

  6. rootDir - 编译文件的目录

  7. outDir - 输出的目录

  8. sourceMap - 代码源文件

  9. strict - 严格模式

  10. module - 默认 common.js 可选 es6 模式 amd umd 等
